API 接口文档

以下接口可用于第三方系统对接，所有请求需携带授权码 key 参数

您的授权码： {{ apiKey }}
GET 获取账号 - 获取一个未使用的账号并标记为已使用
{{ baseUrl }}/api/get-account?key={{ apiKey }}&category_id=分类ID&count=数量
参数说明： category_id (可选) 指定分类， count (可选) 获取数量，默认1
GET 查询账号列表 - 获取账号列表（JSON格式）
{{ baseUrl }}/api/query?key={{ apiKey }}&category_id=分类ID&is_used=true/false
参数说明： category_id (可选) 按分类筛选， is_used (可选) 按使用状态筛选
GET 查询密码 - 根据邮箱查询账号密码
{{ baseUrl }}/api/query/password?key={{ apiKey }}&email=邮箱地址
参数说明： email (必填) 要查询的邮箱地址
GET 统计信息 - 获取账号统计数据
{{ baseUrl }}/api/stats?key={{ apiKey }}
返回：总账号数、已使用数、未使用数、分类数
GET 获取分类列表 - 获取所有分类信息
{{ baseUrl }}/admin/categories?key={{ apiKey }}
返回：分类ID、名称、描述、账号数量

使用示例

# 浏览器直接访问获取账号：

{{ baseUrl }}/api/get-account?key={{ apiKey }}

# cURL 请求示例：

curl "{{ baseUrl }}/api/get-account?key={{ apiKey }}"
