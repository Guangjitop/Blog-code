# 侧边栏模块系统使用指南

## 📋 概述

项目已升级为模块化侧边栏布局，支持：
- ✅ 可折叠的左侧边栏
- ✅ 模块化图标支持
- ✅ 响应式设计
- ✅ 多模块扩展架构

## 🎨 现有功能

### 管理员侧边栏
- **控制台** - 系统概览和统计数据
- **授权管理** - 授权码管理功能

### 用户侧边栏
- **账号管理** - 账号列表和管理
- **获取账号** - 快速获取未使用账号

## 🔧 如何添加新模块

### 1. 在侧边栏组件中添加菜单项

编辑 `frontend/src/components/Sidebar.tsx`：

```typescript
// 1. 导入新图标
import { YourNewIcon } from "lucide-react"

// 2. 在 menuItems 数组中添加新项
const menuItems: MenuItem[] = userType === 'admin' 
  ? [
      { id: 'dashboard', label: '控制台', icon: Home, path: '/admin/dashboard' },
      { id: 'keys', label: '授权管理', icon: Key, path: '/admin/dashboard' },
      // 添加新模块
      { id: 'new-module', label: '新模块', icon: YourNewIcon, path: '/admin/new-module' }
    ]
  : [
      // 用户菜单项...
    ]
```

### 2. 创建新模块页面

在 `frontend/src/pages/` 目录下创建新页面组件：

```typescript
// frontend/src/pages/NewModule.tsx
export default function NewModule() {
  return (
    <div className="min-h-screen bg-background p-8">
      <div className="max-w-7xl mx-auto space-y-8">
        <header>
          <h1 className="text-3xl font-bold">新模块</h1>
          <p className="text-muted-foreground">模块描述</p>
        </header>
        
        {/* 模块内容 */}
      </div>
    </div>
  )
}
```

### 3. 在路由中注册新页面

编辑 `frontend/src/App.tsx`：

```typescript
import NewModule from './pages/NewModule'

// 在 Routes 中添加
<Route path="/admin/new-module" element={
  <MainLayout userType="admin">
    <NewModule />
  </MainLayout>
} />
```

## 🎨 侧边栏特性

### 折叠功能
- 点击顶部折叠按钮可以收起/展开侧边栏
- 折叠状态下悬停图标显示完整标签
- 状态自动保存（基于 React state）

### 主题切换
- 侧边栏底部内置主题切换按钮
- 支持亮色/暗色主题自动切换

### 退出登录
- 侧边栏底部提供退出登录功能
- 根据用户类型（admin/user）自动调用对应API

## 📱 响应式设计

侧边栏已针对不同屏幕尺寸优化：
- 桌面端：完整侧边栏（宽度 256px）
- 折叠状态：仅显示图标（宽度 64px）
- 移动端：可通过按钮切换显示/隐藏

## 🔄 页面布局结构

```
MainLayout (带侧边栏的容器)
├── Sidebar (左侧边栏)
│   ├── Logo 区域
│   ├── 菜单项列表
│   └── 底部操作区
└── Main Content (主内容区)
    └── YourPage (具体页面内容)
```

## 💡 最佳实践

1. **页面内容**：去掉原页面的 `LogOut` 和 `ThemeToggle` 按钮，这些功能已集成到侧边栏
2. **页面宽度**：使用 `max-w-7xl` 作为内容最大宽度
3. **图标选择**：选择语义明确的图标，提升用户体验
4. **菜单层级**：保持菜单项扁平化，避免过深层级

## 🛠️ 故障排查

### 侧边栏未显示
1. 确认已登录到 dashboard 页面
2. 检查浏览器控制台是否有错误
3. 确认 MainLayout 正确包裹页面组件
4. 清除浏览器缓存并刷新

### 图标未显示
1. 确认从 `lucide-react` 正确导入图标
2. 检查图标名称拼写是否正确
3. 确认 Icon 组件类型为 `React.ComponentType<{ className?: string }>`

### 路由不工作
1. 确认 path 与 menuItem.path 一致
2. 检查 App.tsx 中是否添加了对应的 Route
3. 确认页面组件已正确导入
