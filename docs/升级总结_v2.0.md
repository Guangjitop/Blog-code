# 账号管理系统 v2.1.2 升级总结

> 升级日期：2025年12月
> 版本：2.1.2

---

## 一、功能更新概览

### 1. 多风格现代化主题系统

新增 **6 种精心设计的主题风格**，用户可随时切换：

| 主题名称 | 图标 | 风格描述 |
|---------|------|---------|
| 浅色 (Light) | ☀️ | 经典明亮的默认主题 |
| 深色 (Dark) | 🌙 | 护眼深色模式 |
| 海洋蓝 (Ocean) | 🌊 | 清新蓝绿色调 |
| 日落橙 (Sunset) | 🌅 | 温暖橙红渐变 |
| 森林绿 (Forest) | 🌲 | 自然生态绿意 |
| 赛博朋克 (Cyberpunk) | 💜 | 霓虹紫粉科技感 |

**使用方式**：点击页面右上角的调色板图标即可切换主题，选择会自动保存到浏览器。

---

### 2. 批量导入账号功能

用户面板新增"批量导入"功能，支持多种方式快速添加大量账号：

#### 支持的导入格式

**文本粘贴（每行一个账号）**：
```
email@example.com----password123
user@mail.com:mypassword
admin@test.com password456
```

支持的分隔符：
- `----` 四个横线
- `:` 冒号
- 空格/制表符

**JSON 文件导入**：
```json
[
  {"email": "user1@mail.com", "password": "pass1"},
  {"email": "user2@mail.com", "password": "pass2"}
]
```

**TXT 文件导入**：按行解析，格式同文本粘贴

#### 使用步骤
1. 进入用户面板 → 账号列表
2. 点击"批量导入"按钮
3. 选择目标分类（可选）
4. 粘贴文本或点击"导入文件"选择文件
5. 预览解析结果，确认导入

---

### 3. 管理员授权码增强

#### 禁用/启用授权码
- 管理员可临时禁用授权码而非删除
- 禁用后用户无法登录，但数据保留
- 点击电源图标切换状态

#### 自定义授权码
- 创建授权码时可自定义授权码内容
- 留空则自动生成随机授权码
- 自定义授权码需保证唯一性

---

### 4. Toast 通知系统

**移除所有模态弹窗**，采用现代化 Toast 通知：

- 从屏幕顶部滑入显示
- 3 秒后自动淡出消失
- 支持 4 种类型：成功（绿）、错误（红）、警告（橙）、信息（蓝）
- 支持堆叠多条通知

**改进的用户体验**：
- 删除操作改为两步确认（先点击删除按钮，再点击确认）
- 无需弹窗打断操作流程

## 二、技术实现要点

### 前端架构

```
frontend/src/
├── contexts/
│   ├── ThemeContext.tsx    # 主题状态管理
│   └── ToastContext.tsx    # Toast 通知管理
├── components/ui/
│   ├── theme-toggle.tsx    # 主题切换组件
│   └── toast.tsx           # Toast 通知组件
└── pages/
    ├── AdminDashboard.tsx  # 管理员面板（已更新）
    ├── UserDashboard.tsx   # 用户面板（已更新）
    └── ...
```

### 后端 API 新增

| 接口 | 方法 | 描述 |
|-----|------|-----|
| `/api/admin/keys/toggle` | GET | 切换授权码启用/禁用状态 |
| `/api/admin/keys/add` | GET | 新增 `custom_key` 参数支持自定义 |
| `/api/admin/accounts/batch-add` | POST | 批量添加账号 |

### 数据库变更

`auth_keys` 表新增字段：
- `is_enabled` (INTEGER, DEFAULT 1) - 授权码启用状态

## 三、主题配色方案

### CSS 变量定义

每个主题定义以下核心变量：

```css
--background      /* 页面背景色 */
--foreground      /* 主文字颜色 */
--card            /* 卡片背景色 */
--primary         /* 主色调 */
--secondary       /* 次要色调 */
--muted           /* 弱化背景 */
--accent          /* 强调色 */
--destructive     /* 危险/删除色 */
--border          /* 边框颜色 */
--gradient-from   /* 渐变起始色 */
--gradient-to     /* 渐变结束色 */
```

### 主题色彩示例

**Ocean 海洋蓝**：
- 主色：`hsl(200, 80%, 45%)` 深海蓝
- 强调：`hsl(175, 60%, 88%)` 薄荷绿

**Sunset 日落橙**：
- 主色：`hsl(25, 95%, 55%)` 橙红
- 渐变：橙 → 玫红

**Cyberpunk 赛博朋克**：
- 背景：`hsl(270, 30%, 8%)` 深紫黑
- 主色：`hsl(320, 100%, 60%)` 霓虹粉
- 强调：`hsl(260, 60%, 25%)` 电光紫
